Introduction

C++ is a powerful object-oriented programming language that is widely used for developing applications and system software. It is an extension of the C programming language and provides additional features such as classes and objects. This documentation will introduce you to the fundamental concepts and syntax of C++, helping you get started with programming in this language.

What you should already know

Before diving into C++, it is beneficial to have some prior knowledge in the following areas:

  • Basic understanding of programming concepts.
  • Familiarity with the C programming language is advantageous since C++ is an extension of C.

However, it is not mandatory as we will cover the necessary concepts.

C++ and C

C++ is closely related to the C programming language. It includes the entire C standard library, allowing C code to be used in C++ programs. Additionally, C++ introduces new features, such as classes and object-oriented programming, which are not present in C. This combination of compatibility and added functionality makes C++ a versatile language for both C developers and those new to programming.

Hello world

The "Hello world" program is a traditional starting point for learning programming languages. It serves as a simple demonstration of how to output text to the console. In C++, the "Hello world" program can be written as follows:

int main() {
std::cout << "Hello, World!" << std::endl;
return 0;
}

This code requires including input/otput library like below:

#include <iostream>
Variables

In C++, variables are used to store and manipulate data. They hold values of various types such as integers, floating-point numbers, characters and more. Before using a variable, it needs to be declared.

Declaring variables

To declare a variable in C++, you need to specify its type followed by its name. For example:

int age;
double pi;
char letter;
Variable scope

The scope of a variable determines where it is accessible within a program. In C++, variables can have different scopes, such as global scope, local scope, and block scope. Understanding variable scope is crucial for writing well-structured and bug-free programs.

Global variables

Global variables are declared outside of any function and have a global scope. They can be accessed from any part of the program and have a lifetime throughout the program's execution. However, excessive use of global variables is generally discouraged due to potential issues with code maintainability and debugging.

Constants

Constants are values that do not change during the execution of a program. In C++, constants can be declared using the `const` keyword. They provide a way to define fixed values that cannot be modified. You can see exmpale of creating constants below:

const const_name = 5;
Data types

C++ provides a wide range of data types, including primitive types such as integers, floating-point numbers, and characters, as well as more complex types such as arrays, structures, and classes. Understanding different data types and their limitations is essential when working with variables and performing operations on them. Here is an array of chars:

char book_author[] {'S', 't', 'e', 'p', 'h', 'e', 'n', ' ', 'K', 'i', 'n', 'g'};
if else statement

The if else statement in C++ allows you to execute different blocks of code depending on the evaluation of a condition. It provides a way to make decisions and control the flow of your program based on certain criteria.

while statement

The while statement in C++ allows you to repeatedly execute a block of code as long as a condition remains true. It provides a mechanism for looping and iterating until a particular condition is no longer met.

Function declarations

Functions in C++ enable the encapsulation of a group of statements into a reusable block of code. Function declarations define:

  • name
  • return type
  • parameters

It allows function to be called from other parts of the program.
